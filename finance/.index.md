# 个人理财知识版块索引与维护说明

本目录用于沉淀个人理财与长期财务决策相关知识。内容目标是帮助建立清晰的财务认知框架，而不是提供具体投资品推荐。

> 本版块内容仅用于知识学习和方法论整理，不构成投资建议。涉及投资、保险、税务、贷款等事项时，应结合个人情况并咨询具备资质的专业人士。

## 1. 模块定位

* **基础财务秩序**：收入、支出、预算、现金流、应急资金和债务管理。
* **风险管理**：保险、杠杆、流动性风险、市场风险和行为偏差。
* **长期配置框架**：资产类别、风险收益特征、分散配置、再平衡和生命周期管理。
* **决策方法论**：用可验证的数据、清晰的假设和明确的约束辅助财务选择。

## 2. 建议学习路径

1. 个人财务现状盘点：资产、负债、收入、支出和现金流。
2. 应急资金与预算系统：先建立抗风险能力，再讨论投资收益。
3. 保险与风险转移：理解保障目的、保额、免赔、责任范围和常见误区。
4. 债务与杠杆管理：区分消费负债、经营负债、房贷和投资杠杆。
5. 资产类别认知：现金、债券、股票、基金、房产、养老金等。
6. 资产配置与再平衡：根据目标、期限、风险承受能力制定长期策略。
7. 行为金融与决策偏差：识别追涨杀跌、过度自信、损失厌恶等问题。

## 4. 内容边界

* 本版块不提供个股、基金、币种、保险产品的购买建议。
* 涉及市场数据时，必须记录数据日期和来源口径。
* 涉及政策、税务、利率、产品条款时，需要标注时效性，避免把阶段性规则写成永久结论。
* 涉及个人行动建议时，必须说明适用前提、风险和不适用场景。

## 5. 维护约束

* 必须区分事实、假设、观点和行动建议。
* 不使用绝对化收益承诺，不制造确定性。
* 优先讲决策框架和风险边界，再讨论具体工具。
* 新增文档后必须更新本索引的学习路径和文档清单。
